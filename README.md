# README

# 開発言語
* Ruby 3.0.1
* Ruby on Rails 6.0系

# 実行手順
* git clone git@github.com:kikookabe/cathealth.git
* cd ~/cathealth
* bundle install
* rails db:create && rails db:migrate
* rails s
